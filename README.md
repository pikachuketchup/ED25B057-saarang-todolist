A basic ReactJS To Do List app created for the Saarang Web Vertical Selection tasks.

Features:
    Add Tasks
    Remove Tasks
    Mark Tasks as Done
    Responsive Design
    Saarang Theme

Tech Stack Used:
    ReactJS
    JavaScript
    CSS3

Installation Steps:
   1. npm install
   2. npm run dev (or) npm start

For screenshots, refer the folder in github repository
